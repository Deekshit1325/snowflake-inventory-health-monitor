# Inventory Health Monitoring Dashboard (AI for Good)

This project helps organizations proactively monitor inventory levels, detect low-stock risks, and recommend reorder quantities using Snowflake and Streamlit.

## Problem
Stock-outs and overstocking occur because inventory data is fragmented and reactive.

## Solution
A Snowflake-powered system that:
- Monitors daily inventory
- Flags low stock based on lead time
- Recommends reorder quantities
- Visualizes insights via Streamlit

## Tech Stack
- Snowflake (SQL, Dynamic Tables)
- Streamlit (in-Snowflake)
- Python
- Altair

## Features
- Low-stock alerts
- Reorder quantity recommendations
- Heatmap visualization
- KPI summary

## AI for Good Impact
Reduces waste, prevents shortages of essential supplies, and improves operational efficiency.
